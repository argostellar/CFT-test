# CFT-test

Задание выполнялось с помощью расширения Web Server для браузера Chrome

Для запуска программы:
 1. Скачайте расширение из Интернет-магазина Chrome
 2. Запустите расширение
 3. Выберите директорию с файлами текущего проекта используя кнопку Choose Folder
 4. Запустите сервер
 5. Откройте страницу в браузере по указаному ниже URL веб-сервера
